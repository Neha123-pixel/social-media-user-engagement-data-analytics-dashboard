# 📊 Social Media User Engagement Dashboard

## Overview

This project analyzes more than 100,000 social media posts from Facebook, Instagram, YouTube, Reddit, and Twitter using Python and Power BI.

The dashboard helps identify:

- Best posting day
- Highest engagement platform
- Sentiment distribution
- Emotion analysis
- Likes, comments and shares
- KPI tracking

---

## Tools Used

- Python
- Pandas
- Power BI
- SQL
- DAX
- TextBlob
- NLTK

---

## Features

✔ Data Cleaning

✔ Exploratory Data Analysis

✔ Sentiment Analysis

✔ Emotion Detection

✔ Interactive Dashboard

✔ KPI Cards

✔ Slicers

✔ Business Insights
